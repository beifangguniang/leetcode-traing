**问题**

给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的一个字母异位词。

示例 1:

输入: s = "anagram", t = "nagaram"
输出: true
示例 2:

输入: s = "rat", t = "car"
输出: false

说明：
你可以假设字符串只包含小写字母。


**思路**

  1. 根据题意可知，可以利用字母的ASCII表，将各字母出现的次数存储在数组中，ASCII码为索引，值为出现的次数；




2. 比如 s='aabbcd' a 的ASCII码为97，所以在数组中存储：
arry[97]=2；

3. 比对数组，是字母异位返回true，否则返回false。
